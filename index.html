Vercel vaja - CI/CD

To je enostavna spletna stran, objavljena na platformi Vercel. Projekt prikazuje uporabo GitHub repozitorija, Pull Requesta, Preview deploya in Production deploya.

Kaj prikazuje projekt?

Projekt prikazuje osnovni CI/CD proces. Ko spremembo kode shranim, naredim commit in jo pošljem na GitHub, Vercel samodejno izvede deploy.

Preview deploy

Preview deploy nastane iz ločene veje ali Pull Requesta. Uporablja se za testiranje sprememb pred združitvijo v glavno vejo.

Production deploy

Production deploy je javna verzija aplikacije. V tej vaji nastane po združitvi sprememb v vejo main.

Dodatna konfiguracija

V projektu je prikazana dodatna konfiguracija z Vercel environment variable. Environment variables omogočajo, da konfiguracijske vrednosti niso zapisane neposredno v kodo.

  • Platforma: Vercel
  • Repozitorij: GitHub
  • Workflow: branch → commit → push → PR → preview → merge → production

Izdelano za projektno vajo. Tehnologije: HTML, CSS, JavaScript, GitHub in Vercel.
